## English Version

### Learning goals

* Understand the difference between continuous-target prediction and classification
* Get comfortable with the analytic solutions and regularization trade-offs of **linear / Ridge / Lasso** regression
* Use **tree models (Decision Tree, Random Forest, Gradient Boosting)** for non-linear regression
* Practice **feature engineering** — standardization, polynomial features, log transforms
* Evaluate with **MAE / RMSE / R²** and know when each metric is the right one
