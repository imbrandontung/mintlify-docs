### 為什麼 50+ 還在學 ML？

資安攻防的下一波就是 AI。**防守方如果不懂 ML 模型如何運作、如何被欺騙、如何被釘住、如何被竊取**，就會在這一波被甩開。所以我把每天通勤、晚上睡前的零碎時間用來：

* 走 [2nd-ML100Days](https://github.com/imbrandontung/2nd-ML100Days) 結構化挑戰，補齊 ML 工程基本功
* 在 Jupyter Notebook 上手刻 regression / classification / NLP / EDA / scraping / visualization
* 把學到的東西用「資安人能聽懂」的方式整理成 Portfolio
